start() запускает поток по своему пути выполнения, чтобы вызвать метод run()

run() вызывается при реализации отдельной задачи в рамках Runnable

setName() изменяет имя объекта Thread

setPriority(int priority) устанавливает приоритет для потока от 1 до 10

public void interrupt() прерывает этот поток

boolean isAlive возвращает true если поток активен

join(long milisec) текущий поток блокируется до момента завершения второго потока
или не пройдет указанное количество милисекунд
